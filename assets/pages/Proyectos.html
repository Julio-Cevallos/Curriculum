Base de Datos para Finanzas Personales

Desarrollo de un sistema de gestión de finanzas personales mediante una base de datos relacional. El proyecto incluyó el diseño de tablas, relaciones, consultas y reportes que permiten registrar ingresos, gastos y generar análisis financieros personalizados, facilitando la toma de decisiones económicas de manera organizada y eficiente.

Aplicación de Finanzas Personales

Implementación de una aplicación bancaria que permite la administración de cuentas, transferencias y transacciones financieras de forma segura. El proyecto contempló la integración de interfaces de usuario, validaciones, almacenamiento de datos y protocolos de seguridad, proporcionando una experiencia confiable y práctica para la gestión financiera digital.

Enlace: Aplicación_Finanzas

Diseño de Software

Proyecto enfocado en el diseño de software aplicando principios de arquitectura, modelado de clases y patrones de diseño. Incluyó la planificación de módulos, diagramas de flujo y documentación técnica, garantizando la escalabilidad, mantenimiento y eficiencia del sistema desarrollado, así como la correcta implementación de buenas prácticas en ingeniería de software.
