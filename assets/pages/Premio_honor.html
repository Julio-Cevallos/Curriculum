Diseño de Interfaz (UI)

Este certificado acredita la finalización del curso de Diseño de Interfaz (UI) esencial, donde se adquirieron conocimientos fundamentales para crear interfaces de usuario efectivas y atractivas. El programa abarcó principios de diseño visual, tipografía, color, jerarquía de información y patrones de interacción, así como la aplicación práctica de herramientas de diseño para prototipos funcionales. Al completar este curso, el participante está capacitado para desarrollar interfaces intuitivas que mejoran la experiencia del usuario y fortalecen la comunicación visual en productos digitales.

CCNA: Introducción a las Redes

Este certificado acredita la finalización del curso CCNA: Introducción a las Redes de Cisco, donde se adquirieron conocimientos fundamentales sobre redes de datos y conectividad. El programa incluyó conceptos clave de protocolos de red, direccionamiento IP, routers y switches, topologías de red, y fundamentos de seguridad y administración de redes. Al completar el curso, el participante está capacitado para comprender y aplicar los principios básicos de las redes Cisco, facilitando la planificación, configuración y resolución de problemas en entornos de red empresariales.
